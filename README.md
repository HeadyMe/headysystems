# HeadySystems

> HeadySystems platform service

**Status:** Active  
**Last Updated:** March 17, 2026

## Overview

Core platform service for HeadySystems, providing tenant management, billing, analytics, and administrative functions for enterprise customers

## Setup

```bash
git clone https://github.com/HeadyMe/headysystems.git
cd headysystems
npm install
npm start
```

Part of the Heady AI ecosystem, connecting to:
- HeadySystems (headysystems.com) — Enterprise platform
- HeadyMe (headyme.com) — Personal dashboard
- HeadyMCP (headymcp.com) — MCP protocol
- Heady API (api.headysystems.com) — REST API
